# 문서요약 스킬

## 사용 방법

### 1단계: 파일 입력 & 사용량 확인
- **경로 입력**: 파일 절대경로 제공 (예: `C:\Users\...\document.pdf`)
- **드래그&드롭**: 문서를 직접 끌어다 놓기
- **📊 사용량**: 파일 입력 시 자동으로 현재 Groq 사용량 표시

지원 형식:
- **PDF**: 스캔 문서 포함 (자동 OCR)
- **TXT/MD**: 텍스트 파일
- **음성 전사본**: TXT 형식의 전사본

### 2단계: 출력 모드 선택
| 옵션 | 설명 | 사용 시점 |
|------|------|---------|
| **전문 (100%)** | 전체 내용 그대로 | 세부 검토 필요 시 |
| **요약 (50%)** | 핵심 내용 50% | 빠른 파악 필요 시 |
| **요약 (20%)** | 핵심만 20% | 초고속 스캔 필요 시 |

### 3단계: 모델 선택
| 모델 | 특징 | 선택 기준 |
|------|------|---------|
| **Llama 3.1 (권장)** | 최고 정확도 | 정확성 우선 |
| **Llama 2 (토큰절약 50%)** | 빠르고 경제적 | 속도/비용 우선 |

### 4단계: 결과 수신
- **Markdown 포맷**: 제목, 리스트, 강조 등으로 구조화
- **📊 사용량 표시**: 결과 하단에 사용한 토큰 수 및 모델 효율 표시
- **복사 가능**: 결과를 바로 문서에 붙여넣기 가능
- **처리 시간**: 파일 크기에 따라 30초~3분

---

## 설정

**필수**: `.claude/.env` 파일에 Groq API 키 저장
```
GROQ_API_KEY=your_key_here
```

프리 티어 토큰 제한: 초당 30토큰 (충분함)

---

## 주의

- **최대 파일 크기**: 10MB
- **자동 청킹**: 대용량은 내용별로 분할 처리
- **OCR 정확도**: 스캔 문서는 해상도 200dpi 이상 권장
- **개인정보**: 결과는 로컬에서만 처리 (외부 저장 안 함)
- **모델별 효율**: Llama 2는 50% 토큰 절약 (약간 덜 정확)
