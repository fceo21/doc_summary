# 초기 설정

## 1. Groq API 키 발급

1. https://console.groq.com 방문
2. 무료 계정 가입
3. API 키 생성 (이름: `claude-summarizer`)
4. 키 복사

## 2. 환경변수 설정

`C:\cowork\.claude\.env` 파일 열기 (없으면 생성)

```
GROQ_API_KEY=gsk_your_actual_key_here
```

저장 후 Cowork 재시작

## 3. 설치 확인

플러그인 설치 후 `/문서요약` 명령어 실행해서 에러 없으면 OK
