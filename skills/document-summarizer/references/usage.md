# 사용 예시

## 시나리오 1: 긴 PDF 스캔 문서 빠르게 파악

```
파일 경로: C:\Documents\회의록_2024.pdf
출력 모드: 요약 (20%)
모델: Llama 3.1 (정확도 우선)
```

→ 30초 내에 핵심 내용만 마크다운으로 받음

## 시나리오 2: 녹취록 정리 (토큰 절약)

```
파일 경로: C:\Audio\meeting_transcript.txt
출력 모드: 요약 (50%)
모델: Llama 2 (토큰 50% 절약)
```

→ 절반 크기의 마크다운 요약 (더 빠름)
- 참석자 이름
- 주요 결정사항
- 액션 아이템

## 시나리오 3: 계약서 전체 텍스트 추출

```
파일 경로: C:\Contracts\agreement.pdf
출력 모드: 전문 (100%)
모델: Llama 3.1 (최고 정확도)
```

→ OCR 처리된 전체 텍스트 마크다운
- 섹션별 구분
- 링크 가능한 제목
- 하단에 사용량 자동 표시

---

## 팁

**큰 파일 처리 시간:**
- 100페이지 PDF: 1~2분
- 음성 전사본 (50MB): 1분
- 초과 시 자동으로 청크 분할 처리

**마크다운 활용:**
- `# 제목` 레벨로 구조화
- `- 리스트` 형식 자동 생성
- Notion, Word 등에 직접 복사 가능

**모델 선택 팁:**
- 첫 실행: Llama 3.1로 정확도 확인
- 반복 작업: Llama 2로 토큰 절약
- 사용량 체크: 매월 초에 프리 티어 리셋 여부 확인
